# KR-STUDIO 🎬

IDE de escritorio para producción de videos virales de ciberseguridad.

## Características
- 🧠 AI DOMINION (Gemini API) para generación de guiones
- 🌐 Radar OSINT con búsqueda de tendencias en vivo
- 🎯 Targets legales pre-configurados (Vulnerable-by-Design)
- 🔊 TTS con edge-tts + mutagen
- 🎬 Director Dinámico con control dual de terminales (Konsole)
- 📺 Integración con OBS Studio
- 📝 Editor JSON dual (Terminal A / Terminal B)

## Requisitos
- Python 3.10+
- Konsole, xdotool, wmctrl
- OBS Studio (opcional)

## Instalación
```bash
pip install customtkinter google-generativeai edge-tts mutagen obs-websocket-py
```

## Uso
```bash
python kr_studio/main.py
```
