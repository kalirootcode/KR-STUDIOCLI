# Plantillas de Contenido

## Plantilla: Post para Redes Sociales

```
🔥 [GANCHO que atrapa - maxima 10 palabras]
[emojis relevante]
[Valor o conocimiento - 2-3 lineas]
[CTA claro]
#hashtag1 #hashtag2 #hashtag3
```

Ejemplo:
```
🔥 El 90% de los sitios web tienen esta vulnerabilidad

Esta técnica de SQL Injection te puede dar acceso total

Aprende a protegerte HOY
👉 Link en bio

#ciberseguridad #pentesting #hacking
```

## Plantilla: Artículo de Blog

```
TÍTULO:
[Número] + [Gancho emocional] + [Keyword]

INTRO (150 palabras):
- Hook que conecta con el problema
- Por qué este tema importa
- Qué va a aprender el lector

DESARROLLO:
## Subtítulo 1
[Contenido + ejemplos + imágenes]

## Subtítulo 2
[Contenido + ejemplos + imágenes]

## Subtítulo 3
[Contenido + ejemplos + imágenes]

CONCLUSIÓN:
- Resumen de puntos clave
- Próximos pasos
- CTA (lead magnet o venta)

META:
- Título SEO
- Descripción (150 caracteres)
- Keywords
```
