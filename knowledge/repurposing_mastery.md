# Content Repurposing Mastery

## The Repurposing Framework

### Core Principle
```
1 piece of pillar content
        ↓
Extract 10+ pieces of micro-content
        ↓
Distribute across all platforms
        ↓
Drive traffic back to main content
```

---

## From YouTube Video to 10+ Content Pieces

### Original Content
```
Video Title: "Complete Nmap Tutorial for Beginners"
Duration: 20 minutes
Topics covered:
- What is Nmap
- Installation
- Basic scan types
- Advanced techniques
- Real examples
```

---

### Extraction Strategy

#### 1. Shorts/Reels (5-15 clips)
```
Idea 1: "This one nmap command finds open ports in seconds"
- Duration: 30-45 seconds
- Platform: YouTube Shorts, TikTok, Reels

Idea 2: "The mistake everyone makes with nmap -sS"
- Duration: 30-45 seconds
- Platform: YouTube Shorts, TikTok

Idea 3: "Nmap cheat sheet in 60 seconds"
- Duration: 60 seconds
- Platform: All short-form platforms
```

#### 2. Twitter Threads (5-7 tweets)
```
Tweet 1: HOOK
"Most people use nmap wrong. Here's the complete guide 👇"

Tweet 2-6: CONTENT
Thread breakdown of key points from video

Tweet 7: CTA
"Full tutorial on my channel - link in bio"
```

#### 3. LinkedIn Article
```
Title: "Nmap Tutorial: From Beginner to Advanced"
Format: Expanded version with more depth
Length: 1,500-2,000 words
Include: Screenshots, commands, tips
```

#### 4. Blog Post
```
Title: "The Ultimate Nmap Cheat Sheet with Examples"
Format: SEO-optimized article
Length: 2,500-3,500 words
Include: All commands, downloadable PDF
```

#### 5. Newsletter Email
```
Subject: "My complete nmap workflow (with commands)"
Content: Step-by-step guide
CTA: Link to full video
```

#### 6. Instagram Carousel
```
10-12 slides covering key concepts
Slide 1: Hook/title
Slides 2-10: Tips/commands
Slide 11: "Save this for later"
Slide 12: CTA to full video
```

#### 7. Podcast Episode
```
Format: Deep dive conversation
Could be: Solo episode or interview
Length: 30-45 minutes
Topic: Behind the scenes, stories
```

#### 8. PDF Downloadable
```
Title: "Nmap Mastery - Complete Guide"
Include: All commands, cheat sheet
Gate: Email capture
```

#### 9. TikTok Long-form
```
If available: 3-10 minute TikTok version
Different hook than YouTube
More casual tone
```

#### 10. Community Post
```
Discord/Telegram: Key takeaways
Format: Bullet points + link
Engagement: Ask questions
```

## Evergreen vs Trending Content

### Evergreen Content (70%)
```
Repurpose multiple times
Update regularly
Always relevant
Examples:
- How-to tutorials
- Tool comparisons
- Career advice
- Certification guides
```

### Trending Content (30%)
```
Capitalize on trends
One-time push
Time-sensitive
Examples:
- News reactions
- Industry updates
- Viral challenges
- Seasonal topics
```

## Content Atomization Strategy

### Big Content → Small Content
```
PILLAR: Complete course (10 modules)
        ↓
MODULE: 10 individual modules
        ↓
LESSON: 50 individual lessons
        ↓
TIP: 100+ individual tips
        ↓
POST: 500+ social posts
```

### Example: Course Module
```
Module: "Web Application Security"
        ↓
Lessons:
- Introduction to OWASP
- SQL Injection
- XSS Explained
- CSRF Attacks
- Authentication Bugs
        ↓
Each lesson → YouTube video
        ↓
Each video → 5 shorts
        ↓
Each video → Twitter thread
        ↓
Each lesson → Blog post
        ↓
All → Course + free content
```

## Quality Checklist

### Before Repurposing
```
□ Is the original content high quality?
□ Does it have evergreen value?
□ Will it resonate on other platforms?
□ Is the information still accurate?
□ Do I have rights to repurpose?
```

### When Repurposing
```
□ Is it optimized for this platform?
□ Is the format correct?
□ Is the hook appropriate?
□ Does it link back to original?
□ Is the CTA relevant?
□ Does it add value?
```

### After Publishing
```
□ Track analytics per platform
□ Note what formats work best
□ Adjust strategy based on data
□ Double down on winning content
```
